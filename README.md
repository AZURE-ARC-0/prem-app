## Getting Started

### Run the Backend

```bash
# clone the backend repo
git clone git@github.com:premAI-io/prem-box.git
cd ./prem-box/
docker-compose up --build -d prem_controller
```
### Run the app with Tauri

```bash
npm i
npm run tauri dev
```

### Run the app with React

```bash
npm i
npm run start
```
